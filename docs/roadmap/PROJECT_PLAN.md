# VibeWriter

> 카테고리별 블로그 글의 톤앤매너를 분석·학습하고,
> 사용자가 입력한 주제에 대해 일관된 스타일의 블로그 글을 자동 생성한다.

## 1. 개요
### 1.1 프로젝트 목표
블로그 운영의 전체 프로세스를 AI 기반 멀티 에이전트로 자동화한다.

### 1.2 실행 환경 및 운영 방식
  - CLI 실행 모드 지원
    - 터미널에서 Claude Code 기반 즉시 실행
  - 로컬 서버 기반 웹 인터페이스 제공
    - 브라우저를 통해 주제 입력 및 결과 출력

<br><br>

## 2. 기술 스택 및 개발 개발 원칙
### 2.1 기술 스택
  - Language: Python
  - LLM Runtime: Ollama (Local Model)
  - Web Framework: (미정)
  - CLI 실행: Claude Code CLI
  - 크롤링: (미정)
  - 데이터 저장: 로컬 파일 기반 (JSON / Markdown)
### 2.2 개발 원칙
  - 무료 또는 로컬 기반 LLM만 사용
  - 외부 유료 API는 사용하지 않음
  - 로컬 실행을 기본 전제로 설계
  - MVP 우선 개발 후 점진적 확장
  - 모듈 단위 설계 (Agent 분리 구조)


<br><br>

## 3. 개발 Phase
### 3.1 Phase 1 : 스타일 학습 파이프라인 구축
  1. 데이터 입력
     - 블로그 URL 목록 파일 읽기
      - 학습 완료 URL 자동 주석 처리
  2. 콘텐츠 수집 및 전처리
      - URL 크롤링
      - 본문 추출 및 정제
  3. 카테고리 분석
      - 카테고리 분류
      - 카테고리별 글 클러스터링
  4. 톤앤매너 분석
      - 문체 분석
      - 어휘 패턴 분석
      - 구조 패턴 분석
  5. 스타일 가이드 생성 
      - 카테고리별 스타일 가이드 작성
      - 기존 가이드와 중복 제거 후 업데이트
      - humanize 정책을 필수 적용
